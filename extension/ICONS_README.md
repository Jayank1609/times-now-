# Extension Icons

The Chrome extension requires icon files for display. Create the following icon files:

## Required Icons

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Solution

You can create simple placeholder icons using any image editor or online tool:

1. **Design**: Use a professional logo with "NV" or "NewsVerify" text
2. **Colors**: Use the theme colors (#e31837 red and #8a3ffc purple)
3. **Format**: PNG with transparency
4. **Sizes**: Ensure exact pixel dimensions

## Online Icon Generators

- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.canva.com/

## Temporary Solution

If icons are missing, the extension will still work but may show a default icon. You can:
1. Create simple colored squares as placeholders
2. Use any PNG images with correct dimensions
3. The extension will function normally without icons

## Icon Design Suggestion

- Background: Gradient from #e31837 to #8a3ffc
- Foreground: White "NV" or shield/checkmark icon
- Style: Professional, clean, modern
